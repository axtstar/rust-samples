# 🦀🕸️ `sample-rust-wasm`

## 🚴 prerequired

install wasm-pack

> https://rustwasm.github.io/wasm-pack/installer/

### 🛠️ Build

build pkg
```
wasm-pack build
```

npm link to local glocal

pkg link 
```
cd pkg
npm link
```

npm link sample-rust-wasm
```
cd www
npm link sample-rust-wasm
```

start server
```
cd www
npm run start
```
